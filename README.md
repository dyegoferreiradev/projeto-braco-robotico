#🤖 Projeto: Jogo da Velha com Braço Robótico

## 📖 Sobre o Projeto
Este projeto prático tem como objetivo aplicar conhecimentos de programação para a resolução de problemas de robótica de forma prática. O objetivo do projeto é desenvolver um programa que possibilite um braço robótico jogar o Jogo da Velha. As práticas serão no laboratório da Residência em Robótica e IA.

## ⚙️ Funcionamento e Lógica
O sistema atua como a ponte entre a lógica do jogo e o atuador físico:
**Mapeamento:** O tabuleiro é representado por uma matriz 3x3.
**Entrada de Dados:** Seu programa deverá receber as jogadas por meio do terminal.
**Formato Esperado:** O formato de entrada deve ser: 
 `X: (0,1)` [cite: 7]
 `O: (1,0)` [cite: 8]
**Ação Física:** A partir desta entrada, o programa deve mover o braço robótico para realizar uma jogada.
**Fluxo de Jogo:** Essa sequência segue até que alguém ganhe ou que haja empate.
